VirtualEnv: A Platform for Embodied AI Research

Kabir Swain, Sijie Han, Ayush Raina, Jin Zhang, Shuang Li, Michael Stopa, Antonio Torralba
arXiv:2601.07553v2 (last revised Feb 7, 2026)
Paper (PDF) arXiv arXiv HTML BibTeX

VirtualEnv is a high-fidelity Unreal Engine 5 simulator designed for benchmarking LLM-driven embodied agents with rich interactions, procedural scenario generation, and multi-agent collaboration.

Abstract

As large language models (LLMs) continue to improve in reasoning and decision-making, there is a growing need for realistic and interactive environments where their abilities can be rigorously evaluated. We present VirtualEnv, a next-generation simulation platform built on Unreal Engine 5 that enables fine-grained benchmarking of LLMs in embodied and interactive scenarios. VirtualEnv supports rich agent–environment interactions, including object manipulation, navigation, and adaptive multi-agent collaboration, as well as game-inspired mechanics like escape rooms and procedurally generated environments. We provide a user-friendly API built on top of Unreal Engine, allowing researchers to deploy and control LLM-driven agents using natural language instructions. We integrate large-scale LLMs and vision-language models (VLMs), such as GPT-based models, to generate novel environments and structured tasks from multimodal inputs. Our experiments benchmark the performance of several popular LLMs across tasks of increasing complexity, analyzing differences in adaptability, planning, and multi-agent coordination. We also describe our methodology for procedural task generation, task validation, and real-time environment control. VirtualEnv is released as an open-source platform, we aim to advance research at the intersection of AI and gaming, enable standardized evaluation of LLMs in embodied AI settings, and pave the way for future developments in immersive simulations and interactive entertainment.

What VirtualEnv Provides

High-fidelity worlds

Built on Unreal Engine 5 with photorealistic indoor + outdoor settings (offices, retail venues, urban streetscapes).

Large interactive asset library

20,000+ objects with affordances (open / close, grasp, move, use appliances) and realistic physics.

Multi-agent + language

Supports multi-agent collaboration and language-guided planning/interaction via a lightweight Python API.

Language-driven scenario generation

A user prompt is decomposed into sub-goals/puzzles; required objects are instantiated by updating the scene graph and rendering the environment.

Escape room benchmarks

Four levels of escape-room difficulty (from one-step problems to deceptive clues) to stress-test reasoning in dynamic scenes.

Natural-language scene editing

A vLLM converts a command (e.g., “put a key inside the box”) into JSON edits; an interpretation check compares the graph to the rendered view.

Benchmarks highlighted in the paper:

  • Visual realism survey comparing VirtualEnv to OmniGibson, AI2-THOR, VirtualHome, and Habitat (N=31).
  • Reasoning (chain-of-thought) LLM variants outperform base models (reported average +11% task completion).
  • Multi-agent collaboration improves success on multi-step tasks via task allocation and reduced occlusion uncertainty.

Failure mode distribution

The paper analyzes six recurring failure modes for LLM planners in partially observable embodied tasks. The most common is exploration loops (agents repeatedly visiting the same rooms), followed by phantom goals (pursuing non-existent objects).
